# DataMatrix Recognition

Простой инструмент для распознавания DataMatrix кодов из изображений с сохранением результата в текстовый файл.

## Особенности

- Кроссплатформенная поддержка (Windows, Linux)
- Использует современные библиотеки: OpenCV, ZXing-CPP, NumPy
- Работает с Python 3.10+
- Изоляция зависимостей через виртуальное окружение
- Обработка ошибок на русском языке

## Требования

- Python 3.10.11 или выше
- Виртуальное окружение (рекомендуется)

## Установка

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd datamatrixRecognition
```

### 2. Создание виртуального окружения

**Windows:**
```cmd
python -m venv .venv
.venv\Scripts\activate
```

**Linux/macOS:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

## Использование

### Базовый синтаксис

```bash
python datamatrix2.py <input_image> <output_file>
```

### Примеры

```bash
# Распознать DataMatrix из pic.jpg и сохранить результат в result.txt
python datamatrix2.py pic.jpg result.txt

# Использовать изображение из testdata
python datamatrix2.py testdata/pic.jpg output/pic_result.txt
```

### Пример запуска из проекта

```bash
# Windows
.venv\Scripts\python.exe datamatrix2.py pic.jpg output\result.txt

# Linux/macOS
.venv/bin/python datamatrix2.py pic.jpg output/result.txt
```

## Как это работает

1. **Загрузка изображения**: Используется OpenCV для чтения изображения
2. **Предобработка**: Преобразование в оттенки серого и применение пороговой фильтрации OTSU
3. **Распознавание**: Библиотека ZXing-CPP ищет DataMatrix коды
4. **Сохранение результата**: Текст из найденного DataMatrix сохраняется в указанный файл

## Тестовые данные

В директории `testdata/` находятся изображения для тестирования:
- `pic.jpg`, `pic1.jpg`, `pic3.jpg`, `pic4.jpg` — изображения с DataMatrix кодами
- `err.jpg`, `err2.jpg` — изображения без DataMatrix кодов (для тестирования ошибок)

## Результаты

Результаты сохраняются в текстовом файле в формате UTF-8:
- Успешное распознавание: текст из DataMatrix кода
- Ошибка: сообщение об ошибке на русском языке

## Зависимости

- `numpy==2.2.6` — работа с массивами изображений
- `opencv-python==4.13.0.92` — обработка изображений
- `zxing-cpp>=2.2.0` — распознавание DataMatrix кодов

## Обработка ошибок

Программа корректно обрабатывает:
- Изображения без DataMatrix кодов
- Файлы, которые не являются изображениями
- Отсутствие входных файлов
- Проблемы с правами доступа к выходному файлу

## Советы

1. **Качество изображения**: Для лучшего распознавания используйте четкие изображения с хорошим контрастом
2. **Разрешение**: DataMatrix коды должны быть достаточно крупными на изображении
3. **Формат**: Поддерживаются форматы JPG, PNG, BMP и другие, поддерживаемые OpenCV
4. **Виртуальное окружение**: Всегда используйте `.venv` для изоляции зависимостей

## Проблемы и решения

### "ШТРИХКОД НЕ РАСПОЗНАН"

- Проверьте, что изображение содержит DataMatrix код
- Убедитесь, что код достаточно четкий и не поврежден
- Попробуйте другие изображения из `testdata/`

### Ошибки установки

Если возникают проблемы с установкой zxing-cpp:
- Убедитесь, что у вас есть права на установку пакетов
- Проверьте интернет-соединение
- На Linux может потребоваться компилятор C++17 (для сборки из исходников)
